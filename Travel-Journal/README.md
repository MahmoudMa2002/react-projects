# Travel Journal - Location Showcase
A responsive travel journal showcasing visited locations with images, dates, and descriptions.

## Preview
![Project Screenshot](./public/demo.gif)

## Info
**Tech:** React, CSS (Flexbox, Gradients, Transitions), JavaScript  
**Focus:** Component composition, props, responsive design

## Features
- Display multiple travel entries with images
- Google Maps integration for each location
- Responsive layout for mobile and desktop
- Hover effects on cards and images
- Clean data structure with external data file

## Improvements Made
- **Responsive design** – Mobile-friendly layout with media queries  
- **Visual polish** – Enhanced borders, shadows, and refined typography  
- **Image preview** – Click on an image to view it in full size  
- **Improved UX** – Favorited cards are prioritized and displayed first

## What I Learned
- Mapping data to components with props
- Spread operator for passing multiple props
- Component reusability and composition
- Responsive design with flexbox
- CSS transitions and transforms
- External data management
